## For Statement

- Iterating elements
- Do something for each element


```Python
# Measure some strings:
words = ['cat', 'window', 'howdoyouturnthison']
for w in words:
    print(w, len(w))
```

```Bash
cat 3
window 6
howdoyouturnthison 18
```


The range() Function

```Python
lst = range(4) # [0, 1, 2, 3]

for i in lst:
    print(i)
```

```Bash
0
1
2
3
```
